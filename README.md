# OpenUnReID_SPGAN

## Introduction
`OpenUnReID_SPGAN` is a PyTorch-based implementation of [SPGAN](https://arxiv.org/pdf/1711.09020.pdf) for unsupervised domain adaptation (**UDA**) on re-ID tasks.


### Supported datasets
- [x] Market-1501-v15.09.15
- [x] DukeMTMC-reID
- [x] MSMT17_V1
- [x] PersonX_v1
- [x] VehicleID_V1.0
- [x] AIC20_ReID_Simulation
- [x] VeRi_with_plate
